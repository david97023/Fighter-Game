# Fighter-Game

天空之戰 互動遊戲程式設計

開始遊戲，分數零分開始計算分數，攻擊敵機的子彈預設每0.333秒射擊一發子彈，而敵機每0.666秒就會出現一架，總共有三種敵機 : 
1. 小型紅色飛機血量=1，出現機率85%，擊落可得分數500分，掉落寶物機率10%，飛行速度隨機不一定。
2. 中型火紅色飛機血量=3，出現機率10%，擊落可得分數1500分，掉落寶物機率30%，飛行速度先停頓後加速往前衝。
3. 大型紫色飛機血量=10，出現機率5%，擊落可得分數2500分，掉落寶物機率80%，飛行速度固定但會左右彈向前飛行。

成功擊落敵機後，有機率會掉落可以獲得特殊功能的寶物盒，而特殊功能最多暫存兩個，J和K內會有和功能相對應的圖示出現，按壓鍵盤J或K就可以使用其功能，一共有四種不同的功能 :
1. 連續射擊，出現的機率60%，可以提升攻擊敵機子彈的射擊頻率。
2. 凍結敵機，出現的機率20%，會讓在戰場上的敵機暫時停止飛行。
3. 分身術，出現的機率10%，會產生和自己相同的小飛機協助攻擊。
4. 清空戰場，出現的機率10%，會讓在戰場上的敵機瞬間陣亡。
